What is Git?

Git is a version control system; you can think of it as a series of snapshots (commits) of your code base. With the creation of each snapshot you build out a whol application, seeing the growth of a full app. During this time you can make branches and experiment and come back to previous snapshots you took or merge those experiments into your app.

What is GitHub?

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere using those code snapshots that you took during development.
